## Инструкция для установки Windows 1.0 для Oracle VirtualBox 7.0.16_Ubuntu


Для создания и работы с ВМ очень подходит программа Oracle VirtualBox. Выполняем следующие шаги:

1. Сначала заходим в BIOS компьютера, много раз нажимаем на клавишу f2 или f12 или del (в зависимости от компьютера). Навигация по BIOS производится с помощью стрелок.
2. Выбираем меню Configuration (как на фото) 
3. Включаем виртуализацию (в моём случае, она называется Intel Virtual Technology), без неё виртуальная машина не заработает. Рядом с этим пунктом должно быть Enabled.
4. Выходим из BIOS. Переходим к меню Exit. Выбираем пункт Exit with Save changes. Далее высветится окно, точно ли мы хотим выйти. Выбираем Yes. После этого компьютер перезагрузится.
5. Скачиваем ISO образ любой системы (к примеру, Windows 1.0)
6. В VirtualBox нажимаем кнопку "Создать"
    1. Пишем имя системы, выбираем ISO образ нашей системы. Программа должна сама определить систему. Если этого не произошло, то выставляем систему и её тип самостоятельно (Microsoft Windows, Other Windows(x32))
    2. Папку системы выставляем по умолчанию
    3. Далее выставляем размер оперативной памяти, виртуального диска
    4. По необходимости, проводим дополнительную настройку (для корректного отображения UI, виртуальных портов) в соответствующем разделе
7. Запускаем ВМ.
8. С помощью стрелок выполняется навигация. Чтобы выбрать пункт меню, нажимаем Enter. Выбираем "Sfondo 1"
9. Далее выбираем первый пункт "Avvia Windows 1.0"
10. Пользуемся системой!
