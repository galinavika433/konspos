#WinGet #командная-строка #программа #код

**WinGet** - это **командная строка (или интерфейс)**, которая позволяет вам **устанавливать, обновлять, удалять и искать** программы на вашем компьютере гораздо быстрее и удобнее, чем если бы вы делали это вручную.

Вместо того чтобы:
1. Открывать браузер.
2. Искать сайт программы (например, для установки 7-Zip или Google Chrome).
3. Скачивать установочный файл (`.exe` или `.msi`).
4. Запускать его и проходить мастер установки (нажимая "Далее", "Принимаю лицензию" и т.д.).
**WinGet позволяет сделать всё это одной командой.**

### Основные возможности:

1. **Установка программ:** Достаточно набрать `winget install 7zip`, и система сама скачает и установит программу тихо (в фоне) или с минимальным вмешательством.
2. **Поиск программ:** Можно искать приложения по названию или ключевым словам прямо из репозитория (хранилища программ).
3. **Обновление программ:** Это, пожалуй, самая полезная функция. Команда `winget upgrade --all` проверит наличие обновлений для всех программ, установленных через WinGet, и обновит их разом (как это делает центр обновления Windows или App Store на телефоне).
4. **Удаление программ:** Можно удалить программу, зная её точное название или ID (идентификатор).

### Откуда он берет программы?

WinGet подключается к репозиторию (хранилищу), которое называется **Microsoft Community Package Manifest Repository**. По сути, это огромный каталог программ, поддерживаемый сообществом и Microsoft. Там содержатся инструкции о том, откуда скачать ту или иную программу и как её правильно установить.

### Как им пользоваться?

WinGet уже встроен в последние версии Windows 10 и Windows 11. Если его нет, его можно установить из Microsoft Store (приложение "Диспетчер пакетов").

**Примеры команд:**

- `winget search chrome` — найдет все пакеты, связанные с Chrome.

- `winget install Google.Chrome` — установит Google Chrome.

- `winget list` — покажет список всех программ, установленных через WinGet (и не только).

- `winget upgrade` — покажет, для каких программ есть обновления.

- `winget upgrade --all` — обновит все программы.


### Для чего это нужно пользователю?

- **Экономия времени:** Особенно при настройке нового компьютера. Можно написать список программ в файле (скрипте) и одной командой установить всё необходимое (браузеры, архиваторы, плееры), пока вы пьете кофе.

- **Удобство обновлений:** Не нужно помнить, какие программы вы ставили и не пора ли их обновить из-за безопасности. WinGet сделает это за вас.

- **Безопасность:** Меньше шансов скачать программу с подозрительного сайта-двойника, так как WinGet берет ссылки из проверенного источника.
