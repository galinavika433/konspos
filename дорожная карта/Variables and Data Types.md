переменные и типы данных

## 1. переменные

это “коробки” для данных.

```
x = 10name = "просто"
```

## 2. основные типы данных

- `int` → целые числа (10)
- `float` → дробные (10.5)
- `str` → текст ("hi")
- `bool` → True / False

## 3. проверка типа

```
type(x)
```

## 4. преобразование типов

```
int("10")
str(10)
float("3.5")
```
