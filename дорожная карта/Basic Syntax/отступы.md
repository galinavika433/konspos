важная часть для правильного рабочего кода
в питоне нет фигурных скобок `{}`, как в других языках
блок кода создаётся отступом (обычно 4 пробела)

правильно
```python
if True:  
    print("1")  
  
if True:  
    print("2")
```

не правильно
```python
if True:
 print("1")
  print("2")
```

выдает ошибку - `IndentationError`
