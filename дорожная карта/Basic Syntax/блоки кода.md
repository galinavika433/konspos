блок -  это группа инструкций
питон понимает блоки через отступ

например
```python
if True:
    print("1")
    print("2")

print("3")
```

получим
```python
1
2
3
```

но
```python
если True:  
выполнить 1  
выполнить 2  
  
потом выполнить 3
```

то есть
- `print("1")`  внутри `if`
- `print("2")`  внутри `if`
- `print("3")`  уже не внутри `if`

1. выполнился блок  
2. потом программа пошла дальше вниз  
3. дошла до `print("3")`  
4. тоже выполнила
