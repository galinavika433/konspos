Представь, что у тебя есть стопка карточек с числами, и тебе нужно разложить их по порядку - от маленького к большому. Есть несколько способов это сделать. 

- **Пузырьком** — постоянно сравниваем соседей и меняем.

- **Вставками** — как карты в руке, вставляем новую в нужное место.

- **Выбором** — каждый раз ищем самый маленький и ставим в начало.
# 1. Сортировка пузырьком (Bubble Sort)

Представь, что ты ставишь карточки в ряд и начинаешь сравнивать соседние. Если левая карточка больше правой, ты меняешь их местами. Так проходишь весь ряд от начала до конца. Самые большие числа постепенно «всплывают» в конец, как пузырьки воздуха в воде. Потом повторяешь всё сначала, но уже не трогаешь последние числа (они уже на месте). И так до тех пор, пока все числа не встанут по порядку.

```python
def bubble_sort(arr):
    n = len(arr)
    # Проходим по массиву n-1 раз
    for i in range(n - 1):
        swapped = False  # флаг, были ли обмены в этом проходе
        for j in range(n - 1 - i):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        # Если обменов не было, массив уже отсортирован
        if not swapped:
            break
    return arr

# Пример использования
data = [64, 34, 25, 12, 22, 11, 90]
sorted_data = bubble_sort(data)
print("Отсортированный массив:", sorted_data)
```


# 2. Сортировка вставками (Insertion Sort)

Вспомни, как ты раскладываешь игральные карты в руке. Сначала у тебя одна карта (она уже «отсортирована»). Потом ты берёшь следующую и вставляешь её в нужное место среди тех, что уже есть: сравниваешь с соседями и двигаешь карты, чтобы освободить место. Так постепенно все новые карты оказываются на своих местах. В компьютере то же самое: массив делится на отсортированную часть (слева) и неотсортированную (справа). Каждый новый элемент вставляется в отсортированную часть, сдвигая остальные.

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]  # текущий элемент для вставки
        j = i - 1
        # Сдвигаем элементы, которые больше key, вправо
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr

# Пример использования
data = [64, 34, 25, 12, 22, 11, 90]
sorted_data = insertion_sort(data)
print("Отсортированный массив:", sorted_data)
```


# 3. Сортировка выбором (Selection Sort)

Ты ищешь самую маленькую карточку во всей стопке и кладёшь её первой. Потом ищешь следующую самую маленькую среди оставшихся и ставишь второй. И так далее. В компьютере это делается так: на каждом шаге находим наименьший элемент в неотсортированной части массива и меняем его местами с первым элементом этой части. Постепенно слева накапливается отсортированный кусочек.

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n - 1):
        min_index = i
        # Ищем индекс минимального элемента в оставшейся части
        for j in range(i + 1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        # Меняем местами текущий элемент с найденным минимальным
        if min_index != i:
            arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

# Пример использования
data = [64, 34, 25, 12, 22, 11, 90]
sorted_data = selection_sort(data)
print("Отсортированный массив:", sorted_data)
```
