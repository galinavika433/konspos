```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        
        first_op = 0
        second_op = 0

        for i in range(len(nums)):
            first_op = nums[i]
            for j in range(i+1, len(nums)):
                second_op = nums[j]
                if first_op + second_op == target:
                    return [i, j]
```

Этот код на #Python решает задачу Two Sum с помощью грубой силы (brute force), используя два вложенных цикла. Внешний цикл, управляемый переменной i, перебирает каждый элемент списка nums (начиная с индекса 0). Внутренний цикл, управляемый переменной j, начинает перебор элементов, следующих сразу после текущего элемента внешнего цикла (i+1), и продолжается до конца списка. Это гарантирует, что каждая пара чисел проверяется только один раз и что элемент не суммируется сам с собой.
На каждой итерации:
 * first_op получает значение элемента nums[i].
 * second_op получает значение элемента nums[j].
 * Проверяется условие if first_op + second_op == target.
 * Если условие истинно, функция немедленно возвращает список, содержащий индексы этой пары: [i, j].
Метод гарантированно находит решение, поскольку перебирает все возможные уникальные пары в списке. Его временная сложность составляет O(N^2), где N — длина списка nums.
