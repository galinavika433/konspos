```python
import random

ans = ["Боги думают над этим вопром", "Да", "Нет", "Спроси позже", "Скорее да, чем нет", "Скорее нет, чем да",]
 

while True:
	qu = input("Введите вопрос: ")
	if qu.find("exit") != -1:
		print("До встречи!")
		break
	else:
		print(random.choice(ans))
```

Мы вводим вопрос, и программа выдаёт случайный ответ на вопрос из списка. Если нужно завершить программу, мы печатаем exit.
