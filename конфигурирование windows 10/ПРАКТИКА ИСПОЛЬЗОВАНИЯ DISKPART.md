
# ВВЕДЕНИЕ

**DISKPART** — это мощная консольная утилита в Windows для управления дисками, разделами и томами. В отличие от графического интерфейса, она позволяет выполнять задачи гибко и автоматизировать их с помощью скриптов. Ошибки в DISKPART могут привести к потере данных, поэтому практика требует внимательности.
# ПРАКТИЧЕСКИЕ ШАГИ

#### (на примере USB-флешки)

**Подготовка:**

- Вставьте чистый USB-накопитель (данные будут удалены!).
    
- Запустите командную строку от имени администратора.
    
- Введите diskpart (и нажмите Enter).

**Шаг 1: Работа с диском**
`list disk           // Показывает список дисков. Запомните номер вашей флешки (например, 2).
`select disk 2       // ВЫБЕРИТЕ ВАШ ДИСК (обязательно убедитесь, что не выбрали диск C:).
`clean               // Мгновенно удаляет все разделы с выбранного диска.

**Шаг 2: Создание структуры (для загрузочной флешки в UEFI)**
`convert gpt         // Преобразует диск в формат GPT (или convert mbr для BIOS).
`create partition primary size=1000  // Создает основной раздел размером 1000 МБ.

**Шаг 3: Форматирование и активация**
`format fs=fat32 quick   // Быстрое форматирование в FAT32.
`assign letter=Z         // Назначает диску букву Z.
`active                  // Помечает раздел как активный (для загрузки).

**Шаг 4: Завершение**
`exit    // Выход из Diskpart.
`exit    // Закрытие командной строки.
