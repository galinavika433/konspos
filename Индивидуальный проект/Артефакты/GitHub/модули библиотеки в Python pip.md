#модули #библиотека #pip 
1. Для использования встроенных модулей `import module_name` 
2. Для сторонних — сначала установить через `pip install package_name`, потом импортировать 
3. `pip` работает из терминала, а импорт — в коде Python

pip — это менеджер пакетов Python, который позволяет устанавливать дополнительные библиотеки из интернета (обычно из Python Package Index — PyPI).

**1. Подключение встроенных (стандартных) модулей** В Python для использования модуля нужно его импортировать с помощью ключевого слова `import`:

```Python
`import math  # подключение модуля math print(math.sqrt(16))  # использование функции из модуля`
```

Если нужно импортировать из модуля отдельные функции/классы, можно использовать:

```Python
`from math import sqrt print(sqrt(16))`
```

**2. Установка сторонних библиотек через pip** 

```Bash
pip install имя_библиотеки
```

Например, установка библиотеки для работы с HTTP-запросами:

```Bash
pip install requests
```

 Если у вас несколько версий Python, используйте:

```Bash
`python -m pip install имя_библиотеки`
```

или

```Bash
`python3 -m pip install имя_библиотеки`
```
