#библиотека #модули 
Чтобы подключить модули (библиотеки) в Python, используйте ключевое слово `import`. Например:

```Python

`import numpy as np`
```

Если вы хотите импортировать только определенные функции или классы, используйте:

```Python

`from модуль import функция`
```

Для установки библиотеки можно использовать `pip`:

```Bash

`pip install имя_библиотеки`
```
