#библиотека #модули 
Подключать модули (библиотеки) в Python можно с помощью ключевого слова **import**. Вот основные способы: 

1.**Импорт всего модуля**

```Python

`import math print(math.sqrt(16))   # используем функцию через имя модуля`
```


2.**Импорт конкретных функций или классов из модуля**

```Python

`from math import sqrt, pi print(sqrt(25)) print(pi)`
```

3.**Импорт с переименованием (alias)**

```Python

`import numpy as np array = np.array([1, 2, 3])`
```

4.**Импорт всех имен из модуля (не рекомендуется, может затереть существующие имена)**

```Python

`from math import * print(sin(0))`
```

5.**Импорт пользовательских модулей** Если у вас есть файл `mymodule.py` в той же папке:

```Python

`import mymodule mymodule.my_function()`
```

**Важно** - Модули — это файлы `.py` с кодом. - Большинство сторонних библиотек предварительно нужно установить через `pip`, например:

```Bash
`pip install requests`
```

Таким образом, для подключения используется `import` и/или `from ... import ...`.
